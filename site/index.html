BarcodeAPI

Free API for generating 1D / 2D barcodes.

Generate Barcodes Programmatically

Generate a PNG barcodes for your Global Trade Identification Numbers (GTINs) using this API. You can use the Barcode API from your app or your website.

Supports All Formats

The Barcode API supports all formats including EAN-13, EAN-8, Code-128 and QR Codes. If you find unsupported formats, feel free to contact us.

Absolutely Free. Forever.

The Barcode API is free because such a service should exist on the internet. It is simple to support and maintain, and we will never charge for its usage.

API Explorer

Parameter Type Description Value

bcid

 Enum Barcode type. Possible types are ean-13, ean-8, code-128 and QR code

text

 String Global trade identification number (GTIN) to generate the barcode for.

scale

optional

Number The scale factor to change the size of the generated barcode. Possible values are 1-10.

alttext

optional

String The GTIN (or any other text) to display below the barcode

rotate

optional

Enum What rotation to apply to the barcode. Possible values are: N (no-rotation, default), R (90 rotation), L (counter-clockwise 90 rotation), I inverted.
